Task 1 - Student File Management

Objective
Create a simple Student Management File using the File System (fs) module.

Requirements
Create a folder named students

Create a file named students.txt

Add at least 5 student names

Read and display the file contents

Append one new student name

Rename the file to student-list.txt

Delete the file
